# git常见命令

## 常见命令

```bash
# 初始化仓库
git init

# 查看文件状态
git status

# 
git add 文件列表

#
git commit -m 提交信息

# 提交到远程分支
git push 
# 查看提交记录
git log 
```

## 暂时保存更改的代码

```bash
# 存储临时改动
git stash

# 恢复改动
git stash pop
```

## 分支命令

```bash
# 查看分支
git branch

# 创建分支
git branch 分支名称

# 切换分支
git checkout 分支名称

# 删除分支
git branch -d 分支名称
```
